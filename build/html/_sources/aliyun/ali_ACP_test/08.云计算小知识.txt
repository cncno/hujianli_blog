# 云计算通用知识

### 考点整理


1.MapReduce是一种编程模型，用于大规模数据集（大于1TB）的并行运算

	1.概念"Map（映射）"和"Reduce（归约）"，是它们的主要思想；
	2.提供数据划分和计算任务调度功能；
	3.使用了多种有效的错误检测和恢复机制；
	4.顺序处理数据、避免随机访问数据；
	5.平滑无缝的可扩展性；

2.态势感知是一个大数据安全分析平台，他通过机器学习和数据建模发现潜在的入侵和攻击威胁，帮助客户建设自己的安全监控和防御体系，
从而解决因网络攻击导致企业数据泄露的问题，漏洞和弱口令属于弱点。
